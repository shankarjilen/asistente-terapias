📖 Notas / Instrucciones

📌 CÓMO USAR ESTA APP:

  • Presioná "+ Nueva Ficha" para crear un paciente.
  • Completá sus datos personales (nombre, edad, etc.).
  • El paciente debe aceptar el consentimiento (obligatorio).
  • Completá los 4 módulos de diagnóstico.
  • Podés escribir notas adicionales en cada módulo con el botón 📍.
  • Presioná "Guardar Consulta" para registrar la sesión.
  • Usá la 🔍 lupa para buscar y seleccionar pacientes.
  • El icono ⏱️ muestra el historial completo.
  • El botón "PDF Historial Completo" genera un reporte imprimible.

📞 Datos almacenados en la nube con Supabase.
